
# Introvert vs Extrovert Personality Classification

## Project Overview

This project was developed as part of the Kaggle Playground Series competition:

**Predict the Introverts from the Extroverts**

The objective was to build a machine learning model capable of predicting whether a person is an Introvert or Extrovert based on their social behavior, habits, and personality-related features.

## Competition Details

- Competition: Kaggle Playground Series - Season 5, Episode 7
- Evaluation Metric: Accuracy Score
- Platform: Kaggle
- Type: Binary Classification

## Problem Statement

Given information about an individual's social behavior and personality traits, the task is to predict whether the person is:

- Introvert
- Extrovert

The project involved data preprocessing, feature engineering, model training, prediction generation, and Kaggle leaderboard submission.

## Dataset Features

Some of the features included:

- Time Spent Alone
- Stage Fear
- Social Event Attendance
- Going Outside
- Drained After Socializing
- Friends Circle Size
- Post Frequency
- Other behavioral and personality-related attributes

## Data Preprocessing

The following preprocessing techniques were applied:

- Missing value handling
- Categorical feature encoding
- Data cleaning
- Feature preparation
- Dataset transformation for machine learning

## Machine Learning Model

### Artificial Neural Network (ANN)

An Artificial Neural Network (ANN) was implemented to classify individuals as Introverts or Extroverts.

The workflow included:

1. Data preprocessing
2. Feature encoding
3. Train-test preparation
4. ANN model training
5. Prediction generation
6. Kaggle submission creation
